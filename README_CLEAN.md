# C Programs Collection

This repository contains a collection of C programs ranging from basic to intermediate level, focusing on real-world applications and core programming concepts.

## Programs Included

### Encryption and Decryption
- Demonstrates basic encryption and decryption techniques
- Uses character manipulation and simple logic

### Banking System
- Simulates a simple bank system
- Features account creation, deposit, withdrawal, and balance check

### Tax Calculator
- Calculates tax based on income slabs
- Demonstrates conditional statements and logic building

### Ransomware Simulation (Educational)
- Demonstrates file handling and basic encryption concepts
- Intended strictly for educational purposes

## Concepts Covered
- Variables and data types
- Conditional statements
- Loops
- Functions
- Structures
- File handling
- String manipulation

## How to Run
1. Compile the program:
   gcc filename.c -o output
2. Run the executable:
   ./output

## Note
These programs are created for learning and practice purposes. Some programs simulate real-world systems and should be used responsibly.
